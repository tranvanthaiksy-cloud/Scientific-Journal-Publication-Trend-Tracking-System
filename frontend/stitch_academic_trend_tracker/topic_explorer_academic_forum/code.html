Topic Description

Neural Architecture Search (NAS) is a sub-field of automated machine learning (AutoML) that focuses on automating the design of artificial neural networks. NAS has been used to design networks that are on par with or outperform hand-designed architectures.

Currently, research is pivoting from high-computational reinforcement learning methods towards more efficient gradient-based methods and differentiable architectures. Key challenges include the transferability of found architectures and the search efficiency in extremely large spaces.
